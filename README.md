## Equity RAG

### Quick run
- créer l'environnement adéquat
- .env avec votre clé GROQ
- streamlit run app.py à la racine du projet

Ensuite vous téléchargez les fichiers pdf de votre choix (lancer l'indexation des fichiers) et vous pouvez poser les questions de votre choix 


#### Notes

Ce projet est juste un début et un exemple de petit RAG faisable en un rien de temps avec des sources gratuites
